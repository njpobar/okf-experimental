# Tables

List-shaped entities from the [FPL bootstrap-static](/bootstrap-static.md)
payload. Each is one array of records.

* [Players (elements)](elements.md) - one row per player, with pricing, ownership, season totals and per-90 performance stats.
* [Teams](teams.md) - one row per Premier League club, with FPL strength ratings.
* [Gameweeks (events)](events.md) - one row per gameweek, with deadlines, aggregate scoring and popularity stats.
* [Positions (element_types)](element_types.md) - the four positions with squad-selection rules and player counts.
* [Player stat definitions (element_stats)](element_stats.md) - lookup mapping stat machine names to display labels.
* [Chips](chips.md) - playable chip windows and the gameweeks they cover.
* [Phases](phases.md) - scoring phases (Overall plus monthly windows) over gameweek ranges.
